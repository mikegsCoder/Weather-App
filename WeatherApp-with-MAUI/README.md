# Weather-App

:dart:  A simple Weather app built with .NET MAUI. You can enter your city name and the application will provide you with weather data. If such a city does not exist you will be warned that invalid city name is provided. You can switch between Light and Dark theme. You have an option to export weather data in JSON or XML format.

## :hammer_and_wrench: Technologies and Tools used:

- .NET Multi-platform App UI
- XAML
- C#
