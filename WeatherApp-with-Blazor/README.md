# Weather-App

:dart:  A simple Weather app built with Blazor. You can enter your city name and the application will provide you with weather data. If such a city does not exist you will be warned that invalid city name is provided. You can switch between Light and Dark theme. Two different versions of the application are available - Interactive Server and Web Assembly.

## :hammer_and_wrench: Technologies and Tools used:

- Blazor
- C#
- HTML
- Tailwind CSS
